Projeto 1

O projeto tem como objetivo criar uma aplicação web no formato de um site de receitas de baixíssimo custo direcionadas à famílias de baixa renda que sofrem de insegurança alimentar. Muitas famílias sofrem de insegurança alimentar, tendo dificuldade em consistentemente alimentar todos os membros devido a várias razões, uma das principais sendo a pobreza. O projeto busca remediar essa situação através de demonstrar receitas de refeições de baixíssimo custo, acessíveis a famílias de menor renda.
